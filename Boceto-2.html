##Qu es una funcin matemtica?

Una funcin matemtica (tambin llamada simplemente funcin) es la relacin que hay entre una magnitud y otra, cuando el valor de la primera depende de la segunda. De esta manera, toda funcin matemtica consiste en la relacin entre un elemento de un grupo A y otro elemento de un grupo B, siempre que se vinculen de manera nica y exclusiva.

Fuente: https://concepto.de/funcion-matematica/#ixzz6h7EyUXwR

##Que es el dominio de una funcin?

El dominio de una funcin f ( x ) es el conjunto de todos los valores para los cuales la funcin est definida, y el rango de la funcin es el conjunto de todos los valores que f toma.

#Ejemplo

Definimos un dominio x para nuestras funciones

x<-(-10:10)

I. Funcin constante f(x)=2

fx<-rep(5, 21)
plot(x,fx)

II. Funcin idetidad: g(x)=x

gx<-x
plot(x,gx, col="69", type="l")

III. Funcin lineal: h(x)=5x+4 (y=mx+b)

hx<-5*x+10
plot(x,hx, type="o")

IV. Funcin cuadrtica: i(x)=x^2+2

ix<-x^2+2
plot(x,ix, col="red",type="o")

V. Funcin cbica: j(x)=x^3+10

jx<-x^3+10
plot(x,jx, main="Funcin cbica", xlab="Eje x", ylab="Eje y", col="28", type="o")

VI. Funcin racional k(x)=2/x

kx<-2/x
plot(x,kx, main="Funcin racional", sub="Fuente: elaboracin propia", xlab="Eje x", ylab="Eje y", col="85", type="o")

VII. Funcin exponencial: m(x)=7^x+12

mx<-7^x+12
plot(x,mx, main="Funcin exponencial", sub="Fuente: elaboracin propia",xlab="Eje x", ylab="Eje y",col="37", type="o")
