El siguiente trabajo tiene como finalidad mostrar los elementos básicos del funcionamiento del lenguaje de programación. En primer plano tenemos que entender el concepto “funcion” y “dominio”

##¿Qué es una función matemática?

Una función matemática (también llamada simplemente función) es la relación que hay entre una magnitud y otra, cuando el valor de la primera depende de la segunda. De esta manera, toda función matemática consiste en la relación entre un elemento de un grupo A y otro elemento de un grupo B, siempre que se vinculen de manera única y exclusiva.

Fuente: https://concepto.de/funcion-matematica/#ixzz6h7EyUXwR

##¿Que es el dominio de una función?

El dominio de una función f ( x ) es el conjunto de todos los valores para los cuales la función está definida, y el rango de la función es el conjunto de todos los valores que f toma.

#Ejemplo

Definimos un dominio x para nuestras funciones

x<-(-10:10)

I. Función constante f(x)=2

fx<-rep(5, 21)
plot(x,fx)

II. Función idetidad: g(x)=x

gx<-x
plot(x,gx, col="69", type="l")

III. Función lineal: h(x)=5x+4 (y=mx+b)

hx<-5*x+10
plot(x,hx, type="o")

IV. Función cuadrática: i(x)=x^2+2

ix<-x^2+2
plot(x,ix, col="red",type="o")

V. Función cúbica: j(x)=x^3+10

jx<-x^3+10
plot(x,jx, main="Función cúbica", xlab="Eje x", ylab="Eje y", col="28", type="o")

VI. Función racional k(x)=2/x

kx<-2/x
plot(x,kx, main="Función racional", sub="Fuente: elaboración propia", xlab="Eje x", ylab="Eje y", col="85", type="o")

VII. Función exponencial: m(x)=7^x+12

mx<-7^x+12
plot(x,mx, main="Función exponencial", sub="Fuente: elaboración propia",xlab="Eje x", ylab="Eje y",col="37", type="o")
